# [Day 2] 분수의 덧셈

### 기약분수

- 분자와 분모의 공약수가 1뿐이어서 더이상 약분되지 않는 분수

⇒ 기약분수를 구하기 위해서는 분모, 분자의 **최대공약수**를 구해야 한다.

### 유클리드 호제법

- 최대공약수를 구하는 알고리즘
- 2개의 자연수 a, b에 대해서 a를 b로 나눈 나머지를 r이라 하면(단, a > b), **a와 b의 최대공약수는 b와 r의 최대공약수와 같다.**

### 코드로 나타내는 법

1. 재귀

```jsx
function gcd(a, b) {
  return b ? gcd(b, a % b) : a;
}
```

1. 반복

```jsx
function gcd(a, b) {
  let r;
  while (b) {
    c = a % b;
    a = b;
    b = c;
  }
  return a;
}
```
